# AI & Information Technology Presentation

An engaging, informative presentation about Artificial Intelligence and Information Technology designed for an 18-year-old audience.

## Overview

This presentation explores AI from science fiction to modern reality, covering:
- What AI actually is (beyond Hollywood portrayals)
- Modern AI tools (ChatGPT, image generation, etc.)
- AI ethics and societal impact
- Career opportunities and future outlook

**Duration:** 20-30 minutes
**Slides:** 25 slides
**Tone:** Informative, entertaining, and engaging

## Getting Started

### Option 1: PowerPoint (Recommended for beginners)
1. Read `powerpoint-guide.md`
2. Open PowerPoint and create a new presentation
3. Use `presentation-outline.md` to build each slide
4. Add images and video clips (see guide for sources)

### Option 2: Reveal.js (Web-based)
1. Open `index.html` in your web browser
2. Read `revealjs-guide.md` for instructions
3. Complete slides 10-25 following the examples
4. Add media files to `images/` and `videos/` folders
5. Deploy to GitHub Pages or share the HTML file
